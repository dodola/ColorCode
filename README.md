ColorCode
=========

> An android code file reader (一款Android下的代码阅读工具)<br>
一款代码阅读工具，支持c/c++、java、C#、xml、html、css、javascript、latex、ldap、makefile、caml、sql、pascal、perl、php、prolog、python、ruby、scala、shell、tcl等语言的高亮，支持十几种代码样式切换
